这是 `lsof`（List Open Files）命令的完整帮助输出。`lsof` 是一个功能强大的Unix/Linux工具，用于列出当前系统打开的文件和进程信息。

## 参数分类解释

### 基本显示控制
```bash
-?|-h        # 显示帮助信息
-t           # 简洁输出（只显示PID）
-l           # 显示UID数字而非用户名
-n           # 不解析主机名（显示IP地址）
-P           # 不解析端口名（显示端口号）
-v           # 显示版本信息
-V           # 详细搜索模式
```

### 选择条件（Selection）
```bash
-a           # AND逻辑（默认是OR）
-u s         # 按用户选择：-u root（root用户）或 -u ^root（排除root）
-p s         # 按PID选择：-p 123,456 或 -p ^123（排除PID 123）
-c c         # 按命令名：-c ssh 或 -c ^ssh
-g s         # 按进程组选择
```

### 文件类型筛选
```bash
-i           # 网络连接：-i TCP, -i :22（端口22）, -i @192.168.1.1
-U           # Unix域套接字
-N           # NFS文件
-d s         # 按文件描述符：-d 0,1,2（标准输入输出错误）
```

### 目录和文件系统
```bash
+d s         # 显示指定目录下打开的文件
+D D         # 递归显示目录树（较慢）
+x [fl]      # 跨越文件系统或符号链接
```

### 输出格式控制
```bash
-F [f]       # 指定输出字段：-F pcn（PID、命令名、文件名）
-o o         # 文件偏移量显示格式
-s p:s       # 协议状态筛选：-s TCP:ESTABLISHED
-R           # 显示父进程ID
```

### 高级功能
```bash
+|-r [t]     # 重复模式：+r 15（每15秒刷新，直到无文件）/-r 15（持续刷新）
-b           # 避免内核阻塞（可能在某些情况下挂起）
-O           # 无overhead模式（有风险）
-T fqs       # TCP连接状态详细信息
```

## 实用示例

```bash
lsof -i :8090 -P -n 查看大概信息
ps -p 77831  -f  查看详细信息 77831 是具体的pid
kill -9 77831
```

## 重要提示
- **权限**：普通用户只能查看自己有权限的文件
- **root用户**：可以查看系统所有打开的文件
- **性能注意**：`+D` 递归目录可能很慢，`-O` 选项有风险

这个工具在系统调试、性能分析和安全审计中非常有用。
